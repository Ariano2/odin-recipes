We created a Github repository through github's website then copied its SSH url to place it in the 
git clone <SSH url> command. It created a clone of the repository on my local device. 

Inside the project directory odin-recipes, a index.html (our homepage). it was populated with a boilerplate.
It was linked to the web pages dedicated to individual dishes(burger.html and others) which were created 
inside a directory called recipes which in turn is within our main odin-project directory.

Anchor tag was used for the linking and to enhance the accessability of our website to people
deploying screen readers, the lang attribute was added and the alt attribute were added for this 
purpose to the <html> and <img> tags respectively. 

Upon completion al project files were added to the Git staging area using git add command
and they were pushed to the .git directory and github repository using the git commit and 
git push <remote><branch> command.
